# Contributing to LandWatch Kenya

LandWatch maps documented riparian enforcement cases. We accept additions that
expand coverage to new rivers, cities, or enforcement notices.

## How to contribute

- **Add a violation** — open a GitHub Issue with:
  - River name, city, and zone
  - NEMA Gazette notice URL, WRMA enforcement order URL, or county order reference
  - Date of enforcement action
  - GPS coordinates (if publicly available in the notice)
- **Correct an error** — open a GitHub Issue with the correct detail and source URL
- **New river profile** — open a GitHub Issue with WRMA basin survey reference

This repository does **not** accept contributions that:
- Identify individual property owners or residents
- Are based on social media, local rumour, or unverified claims
- Speculate about cause or intent

## Standards

All records must have `verified = confirmed` backed by a cited public source.
GPS coordinates must be derivable from the public enforcement notice.
